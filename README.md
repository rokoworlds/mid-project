# Приложение для Книжного Клуба

Добро пожаловать в приложение для Книжного Клуба! Это веб-приложение создано для того, чтобы участники клуба могли удобно выбирать книги для чтения, оставлять отзывы и оценки, а также предлагать и голосовать за книгу месяца.

## Функционал

### Главная страница

- **Информация о книге месяца**: отображается текущая книга месяца с подробным описанием и автором.
- **Участники клуба**: список всех участников клуба с возможностью просмотра их профилей.
- **Архив книг**: доступ к информации о предыдущих книгах, выбранных в рамках клуба.

### Карточка книги

- **Описание книги**: включает описание, автора и основную информацию о книге.
- **Комментарии участников**: возможность оставлять комментарии и обсуждать книгу.
- **Оценка книги**: участники могут оценивать книги, и средняя оценка отображается на карточке книги.
- **База данных книг**: хранение и управление информацией о книгах (реализовано с помощью базы данных).

### Участники клуба

- **Создание участника**: регистрация новых участников через специальную форму (реализована с использованием системы аутентификации).
- **Личный кабинет**: каждый участник имеет личный кабинет, где может управлять своей информацией и отслеживать свою активность в клубе.

### Голосование за книгу месяца

- **Сбор предложений**: участники могут предлагать книги для голосования.
- **Голосование**: возможность анонимного или персонализированного голосования.
- **Таймер голосования**: установка срока для выбора книги месяца с таймером обратного отсчета.

### Определение места встречи

- **Выбор места встречи**: возможность предложения и голосования за место встречи участников клуба (реализовано с переиспользованием компонентов голосования за книгу).
